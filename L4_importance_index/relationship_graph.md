# Relationship Graph (L4)

This file maps important relationships between concepts, files, and decisions across the brain system.

It supports better retrieval and helps during consolidation when deciding where knowledge belongs.

## Relationship Types

- **Supports** / **Extends**
- **Contradicts** / **Challenges**
- **Depends On**
- **Is Part Of** / **Contains**
- **Led To** (causal)
- **Alternative To**

## Example Format

```markdown
### Concept A
- **Supports**: Concept B (in `key_abstractions.md`)
- **Is Part Of**: Higher abstraction X
- **Contradicts** (weakly): Older assumption in session notes
```

## Usage
- Update during consolidation when new relationships become clear.
- Use this to improve retrieval quality and detect inconsistencies.
- Keep it lightweight — focus on high-value relationships only.
