# Session Notes / Episodic Memory (L3)

This file (or dated files in this folder) captures specific conversations, decisions, experiments, and context from individual sessions.

**Purpose**: Preserve details that may later be promoted to L2 during consolidation, without cluttering higher layers.

## Recommended Practice
- Create dated entries or separate files per significant session when needed.
- During consolidation, review recent entries and promote valuable insights to L2.
- Keep raw or highly specific information here.
